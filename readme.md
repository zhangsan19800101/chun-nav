###  春哥宇宙导航 - 极简个人主页

这是一个基于 **Hugo** 和 **Vercel** 构建的极速、极简、支持实时搜索的个人网址导航页。

**核心特点：**
- **零成本**：利用 GitHub + Vercel 免费托管。
- **零门槛**：全程网页操作，无需写代码，无需安装软件。
- **实时搜索**：支持搜索链接名称、描述甚至网址。
- **自动部署**：修改配置后自动更新，无需手动上传。

---

###  第一步：一键克隆与部署

#### 1. Fork 项目到你的 GitHub
- 在 GitHub 页面右上角点击 **"Fork"** 按钮。
- 等待几秒钟，这个项目就会完整复制到你自己的 GitHub 账号下。

#### 2. 在 Vercel 部署
- 访问 [Vercel 官网](https://vercel.com)，使用 **GitHub 账号登录**。
- 点击 **"Add New Project"**。
- 选择 **"Import Git Repository"**，找到你刚才 Fork 过来的仓库，点击 **"Import"**。

#### 3. 关键设置 (至关重要)
在 Vercel 的配置页面，请务必**手动填写**以下三项，否则无法运行：

| 设置项 | 填写内容 |
| :--- | :--- |
| **Framework Preset** | 选择 **Hugo** |
| **Build Command** | 输入 `hugo --gc --minify` |
| **Output Directory** | 输入 `public` |

- 确认无误后，点击蓝色的 **"Deploy"**。
- 等待约 1 分钟，看到 "Congratulations" 字样，点击 **"Visit"**，你的导航站就上线了！

---

###  第二步：如何修改内容

所有内容的修改都在 GitHub 网页上完成，修改后保存，Vercel 会自动更新网站。

#### 1. 修改导航链接
- 进入你的 GitHub 仓库，找到 `data` 文件夹，点击 **`links.yml`**。
- 点击右上角的 **铅笔图标** (编辑)。
- 按照格式修改或添加链接：

```yaml
- category: 常用推荐  # 分类名称
  links:
    - title: GitHub   # 网站标题
      url: https://github.com # 网址
      icon:         # 图标 (Emoji)
      desc: 代码托管平台 # 描述
```

#### 2. 修改网站大标题 (SEO)
如果你想把“春哥宇宙导航”改成你自己的名字：
- 回到仓库根目录，找到 **`layouts`** 文件夹，点击 **`index.html`**。
- 点击 **铅笔图标** 编辑。
- 找到第 8 行左右的 `<title>` 标签，修改中间的文字：

```html
<!-- 修改前 -->
<title>春哥宇宙导航 - 你的专属链接中心</title>

<!-- 修改后 -->
<title>我的个人主页 - 探索无限可能</title>
```

- 同时，你可以往下拉，找到第 60 行左右的 `<h1>` 标签，修改网页上显示的大标题：

```html
<h1 class="site-title">我的个人主页</h1>
```

#### 3. 保存生效
- 修改完成后，拉到页面最底部，点击绿色的 **"Commit changes"**。
- 回到 Vercel，你会看到它正在自动重新部署，大约 1 分钟后刷新网页即可看到新内容。

###  第三步：绑定自定义域名 (可选)

如果你有自己的域名 (例如 `nav.yourname.com`)，可以绑定到 Vercel。

#### 1. 在 Vercel 添加域名
- 进入你的 Vercel 项目控制台。
- 点击顶部的 **"Settings"** (设置)。
- 在左侧菜单点击 **"Domains"**。
- 在输入框中输入你的域名 (例如 `chunge-nav.com`)，点击 **"Add"**。
- Vercel 会提示你配置 DNS，保留这个页面不要关。

#### 2. 去域名服务商配置 DNS
- 登录你购买域名的网站 (如阿里云、腾讯云、Godaddy、NameSilo 等)。
- 找到 **域名管理** -> **DNS 解析设置**。
- 添加两条记录：

| 记录类型 | 主机记录 (Host) | 记录值 (Value) | 说明 |
| :--- | :--- | :--- | :--- |
| **A** | `@` | `76.76.21.21` | 指向根域名 |
| **CNAME** | `www` | `cname.vercel.com` | 指向 www 前缀 |

*(注：如果你的 Vercel 提示了不同的 IP 地址，请以 Vercel 提示的为准)*

#### 3. 等待生效
- 配置好后，回到 Vercel 的 Domains 页面，它会自动检测。
- 通常几分钟到几小时后，状态会变成 **"Valid Configuration"**，你就可以通过自己的域名访问导航站了。

###  常见问题

- **修改了没反应？**
    - 浏览器有缓存，请尝试 **强制刷新** (Windows: `Ctrl + F5`, Mac: `Cmd + Shift + R`)。
- **Vercel 部署失败？**
    - 检查是否按照“关键设置”填写了 `Build Command` 和 `Output Directory`。
- **图标不显示？**
    - 确保 `static` 文件夹里有一个名为 `favicon.ico` 的文件。
