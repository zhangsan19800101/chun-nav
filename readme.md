## 这都是用千问写的。以下是简易搭建过程也是。

#### 春哥宇宙导航

这是一个基于 Hugo 和 Vercel 构建的极简、极速、支持实时搜索的个人网址导航页。无需编写代码，只需修改配置文件，即可拥有属于自己的导航网站。

#### 核心特性

- **极简设计**：干净清爽的界面，支持深色模式。
- **实时搜索**：输入关键词，即时筛选本页链接、描述甚至网址。
- **零成本部署**：利用 GitHub 和 Vercel 的免费服务，无需服务器。
- **易于维护**：所有链接通过一个 `YAML` 文件管理，修改后自动更新。

#### 快速开始

你不需要在本地安装任何软件，全程在网页上即可完成搭建。

1. **Fork 本项目**
   在 GitHub 页面右上角点击 **"Fork"** 按钮，将本项目复制到你自己的 GitHub 账号下。

2. **部署到 Vercel**
   - 访问 [Vercel 官网](https://vercel.com) 并使用 GitHub 账号登录。
   - 点击 **"Add New Project"**。
   - 选择 **"Import Git Repository"**，找到你刚才 Fork 过来的仓库。

3. **关键设置 (至关重要)**
   在 Vercel 的部署配置页面，请务必手动填写以下三项：
   - **Framework Preset**: 选择 **Hugo**。
   - **Build Command**: 输入 `hugo --gc --minify`。
   - **Output Directory**: 输入 `public`。

   确认无误后，点击 **"Deploy"**。等待片刻，Vercel 会给你一个专属网址，你的导航站就上线了！

#### 如何修改链接

这是整个项目的核心！所有链接都通过一个简单的数据文件管理。

1. **找到配置文件**
   回到你的 GitHub 仓库，找到 `data` 文件夹，点击里面的 `links.yml` 文件。

2. **在线编辑**
   点击右上角的 **小铅笔图标** 进入编辑模式。

3. **修改内容**
   按照文件中的格式，修改、添加或删除你想要的链接。

   **配置示例：**
   ```yaml
   - category: 常用推荐  # 分类名称
     links:
       - title: GitHub   # 网站标题
         url: https://github.com # 网址链接
         icon:         # 图标（支持 Emoji）
         desc: 代码托管平台 # 网站描述
   ```

4. **保存并自动更新**
   修改完成后，拉到底部点击绿色的 **"Commit changes"** 按钮。Vercel 会自动检测到代码变更并重新部署，大约 1 分钟后，刷新你的导航站网页即可看到最新内容。

#### 本地开发 (可选)

如果你希望在本地预览修改效果，可以安装 Hugo 进行调试。

1. **克隆项目**
   ```bash
   git clone https://github.com/你的用户名/你的仓库名.git
   cd 你的仓库名
   ```

2. **启动服务**
   确保已安装 Hugo，在项目根目录运行：
   ```bash
   hugo server
   ```
   然后在浏览器访问 `http://localhost:1313` 即可预览。

#### 项目结构说明

```text
.
├── data/
│   └── links.yml       # 核心数据文件，在这里管理所有链接
├── layouts/
│   └── index.html      # 页面模板，定义了网站的外观和搜索逻辑
├── static/
│   └── favicon.ico     # 网站图标
└── config.yaml         # Hugo 基础配置
```
